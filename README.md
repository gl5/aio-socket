# aio-socket

#### 介绍
自研Java AIO通讯内核<br/>
1. 内核级集群
2. 断线重连功能
3. 优秀的内存池管理
4. 全方面的流量监控（暂未实现）
5. 质简质朴的架构设计
6. 一目了然的常用API封装
7. 无与伦比的半包、粘包处理
8. SSL、TCL加密插件（暂未实现）
9. 心跳检测（自动剔除60s无心跳的连接）
10. 完全调用底层jdk，无任何依赖，无任何jdk底层源码的增强
11. 5G时代、稳定、实时、亚毫秒级、百万并发级的通讯微内核
